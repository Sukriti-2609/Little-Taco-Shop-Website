# 🌮 Little Taco Shop Website (HTML Project)

## 📖 Project Overview
This project is a beginner-friendly multi-page website built using HTML and basic CSS. It represents a fictional restaurant called **“Little Taco Shop”** and demonstrates core web development concepts including page structure, navigation, forms, and semantic HTML.

This project was created as part of my learning journey in web development through a structured tutorial series.

---

## 🎯 Objectives
- Build a multi-page website using pure HTML
- Understand semantic HTML structure
- Implement navigation across multiple pages
- Create tables, forms, and structured content
- Apply basic CSS styling

---

## 🛠️ Technologies Used
- HTML5
- CSS3

## 🌐 Features

### 🏠 Home Page
- Introduction to the Little Taco Shop
- Navigation menu for all pages
- Taco menu with structured table
- “About” section with semantic HTML
- Embedded image and caption

---

### ⏰ Store Hours Page
- Weekly opening hours
- Clean layout with navigation
- Consistent styling across pages

---

### 📮 Contact Page
- Functional contact form
- Input fields with validation

### 🔐 Privacy Policy Page
- Structured content sections
- Information on data usage and privacy
